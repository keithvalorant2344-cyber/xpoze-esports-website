# Xpoze Esports Website

A modern, responsive esports organization website for Xpoze Esports, built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- Dark tier-1 esports visual style with purple, black, and white branding.
- Fully responsive pages for Home, About, Teams, Achievements, Partners, and Contact.
- Animated hero, section reveals, hover states, mobile navigation, loading UI, and reusable components.
- SEO metadata, sitemap, and robots configuration.
- Local visual assets and placeholders for player, partner, and sponsor imagery.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Project Structure

```text
src/app                 App Router pages and SEO files
src/components          Reusable layout, UI, and section components
src/lib                 Shared site data and helpers
public/images           Hero art, logo, sponsor, and player placeholders
```

## Notes

Roster entries, contact details, social links, and sponsor logos are editable placeholders. Replace them with official Xpoze Esports content whenever final assets are available.
